# Проект: Путешествие по России

## О проекте

Отзывчивый одностраничный сайт о путешествиях по России для [Яндекс.Практикум](https://practicum.yandex.ru/) предоставляет информацию о различных местах России, которые стоит посетить.

## Стек

### HTML

- Семантические теги
- Доступность

### CSS

- Медиа запросы и адаптивная верстка
- БЭМ
- Flexbox
- Grid

## Будущие улучшения

- Добавить смену языка
- Добавить анимацию
